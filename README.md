# V-RANGERS — Voxel Rangers Studio LP

V-RANGERS PTE. LTD. のスタジオ紹介ランディングページ。
Studio Introduction (NDA外部向け) の資料をベースに、Web 向けに再構成したもの。

## Stack

- Plain HTML / CSS / JS（フレームワーク非依存）
- Google Fonts (Space Grotesk / Inter / JetBrains Mono)
- GitHub Pages 配信前提

## ディレクトリ

```
.
├── index.html
├── assets/
│   ├── css/style.css
│   ├── js/main.js
│   └── images/         # PPTX から抽出・最適化した素材
├── README.md
└── .gitignore
```

## ローカル確認

```
python3 -m http.server 8000
# → http://localhost:8000/
```
